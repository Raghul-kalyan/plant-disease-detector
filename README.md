# 🌱 Plant Disease Detector

I made this App to help farmers detect diseases earlier than manual methods

---

## 🚀 Features

* 📸 Capture image using device camera
* 🤖 AI-based disease prediction (using Teachable Machine)
* 📊 Shows prediction with confidence level
* 🌍 Works directly in browser (no backend required)

---

## 🧠 How it works

1. User opens the web app
2. Captures an image of a tomato plant leaf
3. The image is processed using a trained machine learning model
4. The model predicts the disease and shows the result
